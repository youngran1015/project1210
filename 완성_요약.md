- `textContent` 사용으로 안전한 업데이트

## 🎯 핵심 개선사항

### Before (이전 버전)
```javascript
// ❌ 문제: 모든 요소를 innerHTML로 변경
element.innerHTML = translations[lang][key];
// → iframe 재생성 → 스크롤 사라짐
```

### After (최종 버전)
```javascript
// ✅ 해결 1: 팝업 열려있으면 스킵
if (lightbox.classList.contains('active')) return;

// ✅ 해결 2: textContent로 안전하게 변경
if (htmlKeys.includes(key)) {
  element.innerHTML = translations[lang][key]; // HTML 필요시만
} else {
  element.textContent = translations[lang][key]; // 기본은 textContent
}
```

## 🐛 문제 해결

### Q1: PDF가 안 보여요
**A**: PDF 파일 경로를 확인하세요
```javascript
contentUrl: 'assets/webtoon1-ep1.pdf' // 경로 확인
```

### Q2: 언어 변경이 안 돼요
**A**: translations.js 파일이 로드되었는지 확인
```html
<script src="translations.js"></script> <!-- head에 있어야 함 -->
```

### Q3: 팝업 중 언어 변경하면?
**A**: 정상입니다! 팝업 열린 상태에서는 번역 스킵됩니다.
팝업을 닫으면 바로 변경된 언어가 적용됩니다.
