# 🌍 다국어 포트폴리오 웹사이트 - 사용 가이드

## ✅ 완료된 작업

### 1. **다국어 데이터 파일 생성** (`translations.js`)
- 한국어 (ko)
- 영어 (en)  
- 일본어 (ja)
- 중국어 (zh)

모든 텍스트가 4개 언어로 번역되어 있습니다.

### 2. **HTML 구조 개선** (`index.html`)
- 언어 선택 버튼 4개 추가 (한국어/English/日本語/中文)
- 모든 번역 가능한 텍스트에 `data-i18n` 속성 추가
- 자동 언어 전환 JavaScript 로직 추가
- 선택한 언어 로컬 저장소에 저장 (재방문 시 유지)

### 3. **CSS 스타일** (`style.css`)
- 기존 CSS에 이미 언어 버튼 스타일이 있었음
- 추가 수정 불필요

## 🚀 사용 방법

### 1. **파일 배치**
세 파일을 같은 폴더에 넣으세요:
- `index.html`
- `translations.js`
- `style.css`

### 2. **웹사이트 열기**
`index.html` 파일을 브라우저로 열면 자동으로 작동합니다.

### 3. **언어 변경**
- 우측 상단에 4개의 언어 버튼이 있습니다
- 클릭하면 즉시 언어가 변경됩니다
- 선택한 언어는 자동으로 저장되어 다음 방문 시에도 유지됩니다

## 🎯 주요 기능

### ✅ **즉시 언어 전환**
- 페이지 새로고침 없이 언어가 바로 변경됩니다
- 부드러운 전환 효과

### ✅ **언어 기억 기능**  
- 선택한 언어가 `localStorage`에 저장됩니다
- 브라우저를 닫았다가 다시 열어도 이전 언어 설정 유지

### ✅ **완전한 번역**
- 상단 네비게이션
- 사이드바 레이블
- 모든 섹션 제목
- 카드 제목 및 설명
- 팝업 내용

## 🔧 커스터마이징 방법

### 1. **번역 내용 수정**

`translations.js` 파일을 열어서 원하는 텍스트를 수정하세요:

```javascript
ko: {
  hero_title: "Anne",  // 이 부분을 원하는 텍스트로 변경
  hero_subtitle: "Visual Story by Anne",
  // ...
}
```

### 2. **새로운 콘텐츠 추가**

새 카드나 섹션을 추가할 때:

**Step 1: HTML에 `data-i18n` 속성 추가**
```html
<h3 data-i18n="new_card_title">새 카드 제목</h3>
```

**Step 2: translations.js에 번역 추가**
```javascript
ko: {
  new_card_title: "새 카드 제목",
  // ...
},
en: {
  new_card_title: "New Card Title",
  // ...
}
```

### 3. **새 언어 추가**

`translations.js`에 새 언어 섹션을 추가하고, HTML에 버튼을 추가하세요:

```html
<button class="lang-btn" data-lang="fr">Français</button>
```

```javascript
const translations = {
  ko: { /* ... */ },
  en: { /* ... */ },
  fr: {
    hero_title: "Anne",
    // 프랑스어 번역...
  }
};
```

## 🐛 문제 해결

### **언어가 변경되지 않아요**
1. `translations.js` 파일이 같은 폴더에 있는지 확인
2. 브라우저 콘솔(F12)에서 에러 메시지 확인
3. `data-i18n` 속성이 제대로 설정되어 있는지 확인

### **일부 텍스트만 번역돼요**
- `data-i18n` 속성이 빠진 요소가 있는지 확인
- `translations.js`에 해당 키가 모든 언어에 있는지 확인

### **저장된 언어가 유지되지 않아요**
- 브라우저의 로컬 저장소가 활성화되어 있는지 확인
- 시크릿 모드에서는 저장되지 않습니다

## 📝 추가 작업 필요 사항

### **팝업 내용 다국어화**
현재 팝업의 상세 설명은 다국어화되지 않았습니다. 필요하면:

1. `popupData` 객체를 언어별로 분리
2. 팝업 열 때 현재 언어에 맞는 데이터 사용

예시:
```javascript
const popupData_ko = { /* 한국어 */ };
const popupData_en = { /* 영어 */ };

function openPopup(popupId) {
  const data = window['popupData_' + currentLang][popupId];
  // ...
}
```
